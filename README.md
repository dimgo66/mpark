# Михаил Пак — сайт-портфолио

Информационный сайт, посвящённый творчеству российского корейского писателя и художника **Михаила Тимофеевича Пака** (р. 1949).

---

## ✅ Реализованные функции

### Страницы и секции (одностраничный сайт)
- **Hero-секция** — шапка с именем, цитатой и кнопками навигации
- **Биография** — факты о жизни, портрет, полный список литературных премий
- **Живопись** — стиль, факты о выставках, цитата из критической статьи
- **Галерея** — изображения из открытых источников + YouTube-ролики с выставок
- **Книги** — три вкладки: «Романы и повести», «Рассказы», «На корейском»
- **Журнал «Вторник»** — 10 ссылок на конкретные публикации автора
- **Купить книги** — ссылки на OZON (Том 1 и Том 2 избранных сочинений), Azon.market, RoyalLib, Kyobo Book Store
- **Полезные ссылки** — карточки с 8 ресурсами (Корё Сарам, LiveLib, НГ, YouTube и др.)
- **Подвал** — навигация, внешние ссылки, копирайт

### Техническое
- Адаптивный дизайн (mobile-first, брейкпоинты 400/680/900/1100px)
- Липкий заголовок с эффектом размытия при скролле
- Мобильное меню (гамбургер с анимацией)
- Лайтбокс для галереи картин
- Вкладки (tabs) для секции «Книги»
- Плавная прокрутка к секциям
- Анимация появления элементов при скролле (IntersectionObserver)
- Подсветка активного пункта меню при скролле
