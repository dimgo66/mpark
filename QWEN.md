- Рабочее правило для hero-секции: использовать <img> с object-position для точного позиционирования портрета. Для сайта Михаила Пака: object-position: center 35% опускает портрет ниже, чтобы глаза были лучше видны. Меньший процент = ниже изображение, больший процент = выше.
- Рабочее правило: НЕ отправлять изменения на GitHub (git push) без явной просьбы пользователя. Сначала делать коммит, показать пользователю, и только после просьбы "отправь на GitHub" делать push.
