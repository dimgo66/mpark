# Мониторинг и отчетность SEO-показателей

## Обзор
Документ описывает систему мониторинга эффективности SEO-оптимизации для сайта Михаила Пака. Включает ключевые показатели, инструменты и периодичность проверок.

---

| Показатель | Целевое значение | Инструмент проверки |

## 2. Инструменты мониторинга

### Обязательные инструменты
1. **Google Search Console**
   - Мониторинг индексации
   - Анализ запросов
   - Исправление ошибок
   - Проверка Core Web Vitals

2. **Google Analytics 4**
   - Отслеживание трафика
   - Поведение пользователей
   - Конверсии
   - Демография

3. **Яндекс.Вебмастер**
   - Статистика ТИЦ
   - Внешние ссылки
   - География посетителей
   - Ошибки сканирования

4. **Яндекс.Метрика**
   - Вебвизор
   - Цели и конверсии
   - Карта кликов

### Дополнительные инструменты
5. **Google PageSpeed Insights**
   - Анализ скорости
   - Рекомендации по оптимизации

6. **Screaming Frog SEO Spider**
   - Технический аудит
   - Поиск битых ссылок
   - Анализ метатегов

7. **Ahrefs / Semrush**
   - Анализ позиций
   - Исследование ключевых слов
   - Анализ конкурентов
   - Мониторинг обратных ссылок

8. **Google Alerts**
   - Упоминания имени автора
   - Упоминания сайта

## 3. Периодичность проверок

### Ежедневно
- Проверка доступности сайта
- Мониторинг критических ошибок (500, 404)
- Уведомления от Search Console

### Еженедельно
1. **Технический аудит**
   - Проверка скорости загрузки
   - Мониторинг Core Web Vitals
   - Проверка индексации новых страниц

2. **Анализ позиций**
   - Проверка позиций по 10 ключевым запросам
   - Анализ изменений в ТОП-10

3. **Анализ трафика**
   - Основные источники трафика
   - Ключевые запросы
   - Страницы с наибольшим трафиком

### Ежемесячно
1. **Полный SEO-отчет**
   - Динамика органического трафика
   - Изменения позиций
   - Анализ конверсий
   - Отчет по обратным ссылкам

2. **Контент-аудит**
   - Обновление устаревшего контента
   - Добавление новых материалов
   - Оптимизация существующих страниц

3. **Анализ конкурентов**
   - Изменения в стратегиях конкурентов
   - Новые возможности для продвижения

### Квартально
1. **Полный технический аудит**
   - Глубокий анализ всех технических аспектов
   - Проверка соответствия современным стандартам

2. **Стратегический пересмотр**
   - Корректировка семантического ядра
   - Обновление дорожной карты
   - Постановка новых целей

## 5. Автоматизация мониторинга

### Настройка уведомлений
1. **Google Search Console**
   - Уведомления о критических ошибках
   - Уведомления о ручных действиях
   - Уведомления о проблемах с безопасностью

2. **Google Analytics 4**
   - Уведомления о резких изменениях трафика
   - Уведомления о достижении целей

3. **Мониторинг uptime**
   - Настройка оповещений о недоступности сайта
   - Использование UptimeRobot или аналоги

### Автоматические отчеты
1. **Google Looker Studio**
   - Создание дашборда с ключевыми метриками
   - Автоматическая рассылка отчетов

2. **Google Sheets + API**
   - Автоматический сбор данных о позициях
   - Визуализация динамики

3. **Скрипты для автоматизации**
   - Скрипт проверки скорости
   - Скрипт проверки индексации
   - Скрипт мониторинга позиций

## 6. Действия при ухудшении показателей

### Падение трафика
1. **Анализ причин**
   - Проверка обновлений алгоритмов
   - Анализ изменений на сайте
   - Сравнение с конкурентами

2. **Немедленные действия**
   - Проверка технических ошибок
   - Анализ потерянных позиций
   - Обновление контента

### Падение позиций
1. **Анализ запросов**
   - Какие запросы потеряли позиции
   - Изменения в SERP
   - Контент конкурентов

2. **Корректирующие действия**
   - Оптимизация страниц
   - Улучшение контента
   - Увеличение релевантности

### Технические проблемы
1. **Приоритизация**
   - Критические ошибки (500, 404)
   - Проблемы со скоростью
   - Ошибки индексации

2. **Исправление**
   - Немедленное устранение критических проблем
   - Планирование исправления остальных
