# Комплексная SEO-стратегия для сайта Михаила Пака

## Обзор проекта
**Сайт:** Портфолио писателя и художника Михаила Тимофеевича Пака  
**Цель:** Увеличение органического трафика из Google и Яндекс, повышение узнаваемости автора, продвижение книг и художественных работ  
**Текущее состояние:** Новый сайт без аналитики, одностраничный лендинг с качественным контентом

---

## 1. Анализ текущего состояния

### Сильные стороны
- Качественный уникальный контент о нишевой персоне
- Адаптивный дизайн и хорошая производительность
- Наличие структурированных данных (биография, премии, галерея)
- Внешние ссылки на авторитетные ресурсы (Корё Сарам, OZON, YouTube)

### Проблемные области
- Отсутствие robots.txt и sitemap.xml
- Нет аналитики и вебмастеров
- Ограниченное семантическое ядро
- Одна страница для всего контента
- Изображения загружаются с внешних источников (риск 404)

## 3. Техническая оптимизация

### Приоритет 1 (Критично)
1. **Создать robots.txt**
   ```
   User-agent: *
   Allow: /
   Sitemap: https://dimgo66.github.io/mpark/sitemap.xml
   ```

2. **Генерация sitemap.xml**
   - Включить главную страницу
   - Приоритет 1.0, частота обновления weekly
   - Добавить изображения с описаниями

3. **Оптимизация загрузки изображений**
   - Локальное хостинг критичных изображений
   - Оптимизация форматов (WebP + fallback)
   - Добавление lazy loading для всех изображений

4. **Ускорение сайта**
   - Минификация CSS/JS
   - Кэширование через .htaccess или GitHub Pages настройки
   - Предзагрузка шрифтов

### Приоритет 2 (Важно)
5. **Микроразметка Schema.org**
   - Person (для автора)
   - Book (для каждой книги)
   - Painting (для картин)
   - Event (для выставок)
   - BreadcrumbList

6. **Канонические URL и метатеги**
   - Проверка дублей
   - Оптимизация title/description для каждого раздела

7. **Мобильная оптимизация**
   - Проверка Core Web Vitals
   - Улучшение CLS (Cumulative Layout Shift)

### Приоритет 3 (Дополнительно)
8. **HTTPS и безопасность**
   - Уже есть (GitHub Pages)
   - Добавить security headers

9. **Международная SEO**
   - hreflang для корейской аудитории
   - Альтернативные версии на корейском

## 4. Контентная стратегия

### Текущий контент (оптимизация)
1. **Главная страница**
   - Добавить H2-H4 с ключевыми словами
   - Расширить текстовый контент в каждом разделе
   - Добавить FAQ раздел

2. **Биография**
   - Структурировать по годам
   - Добавить внутренние ссылки на книги и картины

3. **Галерея**
   - Оптимизировать alt-тексты с ключевыми словами
   - Добавить описания к каждой картине
   - Создать отдельные страницы для популярных картин

### Новый контент (расширение)
4. **Блог/Статьи**
   - "История корейской диаспоры в России"
   - "Обзор книг Михаила Пака"
   - "Анализ художественного стиля"
   - "Интервью с автором"

5. **Отдельные страницы**
   - Страница для каждой книги с описанием, отзывами, ссылками на покупку
   - Страница для каждой картины с историей создания
   - Страница "Выставки" с хронологией
   - Страница "Премии и награды" детализированная

6. **Мультимедиа**
   - Видео-интервью с субтитрами
   - Аудио-версии рассказов
   - PDF-версии произведений

## 5. Ссылочная масса (Link Building)

### Внутренние ссылки
- Создать перелинковку между разделами
- Добавить "похожие материалы" в конце каждого раздела
- Хлебные крошки для навигации

### Внешние ссылки (получение)
1. **Гостевые посты**
   - Литературные блоги
   - Художественные порталы
   - Культурные издания

2. **Каталоги и справочники**
   - Каталоги писателей
   - Справочники художников
   - Энциклопедии корейской диаспоры

3. **Социальные сигналы**
   - Продвижение в литературных сообществах
   - Упоминания в профессиональных группах

4. **Партнерства**
   - Сотрудничество с "Корё Сарам"
   - Ссылки от журнала "Вторник"
   - Упоминания на сайтах книжных магазинов

### Мониторинг ссылок
- Отслеживание упоминаний (Google Alerts, Brand24)
- Работа с битыми ссылками
- Обновление ссылочного профиля

## 6. Особенности Google и Яндекс

### Google (международный акцент)
- **Акцент на E-A-T** (Expertise, Authoritativeness, Trustworthiness)
- Подтверждение авторства через Google Authorship
- Структурированные данные для картинок в поиске
- Оптимизация для мобильных устройств
- Важность Core Web Vitals

### Яндекс (российский акцент)
- **Региональность** - указать Москва/Россия
- **Тематический индекс цитирования (ТИЦ)**
- Участие в Яндекс.Дзене
- Интеграция с Яндекс.Картинками
- Учет морфологии и синонимов

### Общие рекомендации
- Естественные тексты без переоптимизации
- Регулярное обновление контента
- Социальные сигналы (поделиться в соцсетях)

## 7. Аналитика и мониторинг

### Инструменты для внедрения
1. **Google Analytics 4**
   - Отслеживание трафика по источникам
   - Поведение пользователей
   - Конверсии (клики на покупку книг)

2. **Яндекс.Метрика**
   - Вебвизор для анализа поведения
   - Цели и конверсии
   - Демография аудитории

3. **Google Search Console**
   - Мониторинг индексации
   - Анализ запросов
   - Исправление ошибок

4. **Яндекс.Вебмастер**
   - Статистика ТИЦ
   - Внешние ссылки
   - География посетителей

### Ключевые показатели (KPI)
- Органический трафик (месячный рост 20%)
- Позиции по целевым запросам (ТОП-10 за 3 месяца)
- Индекс цитирования (ТИЦ до 10 за 6 месяцев)
- Конверсии в клики по партнерским ссылкам
- Время на сайте (> 2 минут)

## 10. Риски и митигация

### Риск 1: Низкая частота запросов
**Митигация:** Фокус на длинном хвосте, создание образовательного контента

### Риск 2: Конкуренция с крупными порталами
**Митигация:** Узкая специализация, экспертный контент, персонализация

### Риск 3: Изменения алгоритмов
**Митигация:** Следование best practices, регулярное обновление, диверсификация

### Риск 4: Ограниченные ресурсы
**Митигация:** Приоритизация, фокус на высокоэффективных действиях, автоматизация

## Заключение

Данная стратегия обеспечивает комплексный подход к SEO-продвижению сайта Михаила Пака, учитывая особенности как Google, так и Яндекс. Ключевые элементы успеха:

1. **Техническая безупречность** - основа для индексации
2. **Качественный уникальный контент** - привлечение целевой аудитории
3. **Постепенное наращивание авторитета** - через ссылочную массу
4. **Постоянный мониторинг и адаптация** - к изменениям алгоритмов

Рекомендуется начать с технической оптимизации и постепенно переходить к контентному расширению и активному link building.
